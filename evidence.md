## Evidence Integrity

- All screenshots were captured automatically by the audit script using Canvas rendering — no post-processing.
- The master dataset SHA-256 (8382e45f8cdf4d439a8866669d15e6f4be543f4b926fb64c67e09d9da7d6b2db) was recorded at verification time (Step 18, EC-25) and can be recomputed: `shasum -a 256 results/merged/plinko-master.json`
- The verification code was run against this exact dataset; artifacts in `outputs/` were generated from this run.
